# Hello World

This is a simple greeting skill that demonstrates how skills work.

## Usage

When invoked, this skill will:
- Greet the user with a friendly welcome message
- Include the current date and time
- Provide an encouraging message about using AI agentic tools

## Example

The skill generates a personalized greeting that includes:
- A friendly welcome that incorporates the user's name (if provided)
- The current date and time
- A one-sentence encouraging message about learning AI agentic tools

## Parameters

- **name** (optional): The name of the person to greet. If not provided, uses "there"

## Output

A friendly greeting message formatted as markdown, including:
- Personalized welcome
- Current date and time
- Encouraging message about AI development
